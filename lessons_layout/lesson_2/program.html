Описание курса

В первой части курса вы познакомитесь со стандартами Web 2.0, науситесь работать с макетами и форматировать код, освоите адаптивную верстку, а по итогу — сможете создавать быстрые и удобные сайты, которые точно понравятся пользователям.

Вы поймёте назначение основных тегов HTML, сможете без усилий стилизовать и размещать элементы с помощью CSS и сверстаете свой первый макет.

Вторая часть курса сделает из вас настоящего мастера. Вы изучите сборщики проектов, научитесь ерстать email-рассылки, разберетесь с анимацией на сайте.

Помимо этого, вы сверстаете фрагмент из интернет-магазина - главную раницу, страницу каталога и страницу товара!
